Progress Tracker EDN

Progress Tracker – Escola da Nuvem (EDN)

O Progress Tracker da Escola da Nuvem (EDN) é uma aplicação web criada para organizar e acompanhar o progresso dos alunos nos Labs, Knowledge Checks (KCs) e Competências Profissionais (CPs) da formação em cloud computing na AWS.

O site centraliza essas informações em uma interface simples e visual, permitindo que o aluno saiba rapidamente o que já foi concluído e o que ainda precisa avançar.

Este é um projeto pessoal, desenvolvido com foco em clareza, usabilidade e automação, utilizando AWS CloudFront para distribuição, CodePipeline para deploy automatizado e Route 53 para gerenciamento de domínio. Ele demonstra a aplicação prática de conceitos de front-end e arquitetura cloud em um cenário real de acompanhamento educacional.

Portfólio Pessoal

Portfólio Pessoal com Deploy Automatizado na AWS

Este é o portfólio que você está acessando neste momento, desenvolvido para apresentar meus projetos, experiências e certificações de forma clara e objetiva. Além do front-end, o foco do projeto está na infraestrutura em nuvem, utilizando serviços da AWS para garantir performance, automação e confiabilidade.

O site é distribuído via Amazon CloudFront, conta com deploy automatizado utilizando AWS CodePipeline, gerenciamento de domínio com Route 53 e infraestrutura definida por CloudFormation, permitindo atualizações rápidas, seguras e reproduzíveis.

Mais do que uma vitrine, este portfólio também funciona como um projeto prático de Cloud Computing, onde aplico conceitos reais de automação, entrega contínua e arquitetura escalável em um ambiente em produção.

    AWS • CloudFront • S3 • CI•CD • Serverless

    Como este site funciona

    Este portfólio é um site estático hospedado na AWS com deploy automatizado e distribuição global.

    • S3 armazena os arquivos do site (HTML/CSS/JS/imagens).
    • CloudFront entrega o site rápido no mundo todo (CDN).
    • CodePipeline faz deploy automático a cada commit noF GitHub.
    • CloudFormation define a infra como código.
    • Route 53 gerencia DNS e domínio.
    • API Gateway + Lambda processam o formulário de contato com um backend serverless, sem necessidade de servidores.
    • SES envia os e-mails do formulário usando domínio verificado, reduzindo chance de spam.
    S3 CloudFront CodePipeline CloudFormation Route 53 API Gateway Lambda SES
